1. Bienvenue dans BYT3LAB Builder 👋

BYT3LAB Builder n'est pas un constructeur de pages classique qui pollue votre système avec des codes inutiles. C'est un générateur de thème natif.

À l'aide de cet outil, vous générez un thème 100% pur (code PHP/CSS/HTML traditionnel) de manière visuelle depuis l'administration de WordPress. Une fois votre thème exporté en .zip, il fonctionnera de manière totalement autonome sur n'importe quel autre site, sans nécessiter le Builder !

💡 Astuce : Ce constructeur repose sur la théorie des Composants (Lego). Vous créez des blocs (Header, Hero, Footer), puis vous les empilez sur des Pages.

2. Votre Premier Thème

  1. Rendez-vous dans le menu BYT3LAB Builder > Thèmes dans votre panneau WordPress.
  2. Remplissez le formulaire : Nom du thème, description, et auteur. Choisissez un "Modèle de départ" si vous désirez des présélections.
  3. Cliquez sur Créer.
  4. Allez dans le menu classique de WordPress : Apparence > Thèmes, trouvez votre thème et cliquez sur Activer.

3. Créer des Composants

Un composant est une "brique" visuelle réutilisable. (ex: Un menu, un carrousel, un pied de page).

  1. Allez dans BYT3LAB Builder > Composants.
  2. Sélectionnez votre thème dans le menu déroulant.
  3. Donnez un nom (ex: header ou hero-banner).
  4. Une fois créé, vous pouvez éditer directement son code HTML, CSS, et JS depuis l'onglet Éditeur de l'outil ! N'oubliez pas les bonnes pratiques de nommage.

4. Ajouter vos styles généraux (Assets)

Certains styles (comme la police d'écriture globale ou un framework web) doivent être accessibles partout.

  • Allez dans BYT3LAB Builder > Assets.
  • Vous pouvez télécharger un fichier `.css` (ex: `main.css`) ou `.js` (ex: `app.js`).
  • Ils seront disponibles pour être injectés prioritairement sur vos pages.

5. Créer et Assembler une Page (La magieopère)

Il est temps d'assembler vos briques (Composants) et vos couleurs (CSS) pour créer une vraie page web !

  1. Allez dans BYT3LAB Builder > Pages.
  2. Donnez un nom à votre page, par exemple "Accueil" (home).
  3. Dans la boîte de sélection Assets CSS, cliquez pour choisir les feuilles de style globales à utiliser pour la page.
  4. Dans la boîte de sélection Composants (le point le plus important), sélectionnez vos blocs. Utilisez CTRL (ou CMD sur Mac) pour en choisir plusieurs.
  5. Utilisez les boutons "Monter" / "Descendre" pour changer l'ordre exact dans lequel ils s'afficheront sur votre site !
  6. Cliquez sur Mettre à jour la page.
✅ C'est prêt ! Le constructeur vient de générer un code parfait, optimisé pour le référencement (SEO). Vous pourrez toujours revenir éditer cette page pour rajouter un nouveau composant plus tard, le formulaire se pré-remplira avec votre configuration actuelle !

6. Paramètres : Dire à WordPress qui est qui

Vous avez créé des pages incroyables, mais WordPress ne sait pas laquelle utiliser en page d'accueil !

  1. Allez dans BYT3LAB Builder > Paramètres.
  2. Dans Pages spéciales du thème, assignez vos créations aux rôles de WordPress :
    • Page d'accueil (front-page) : Sélectionnez la page que vous affichez à l'ouverture du site.
    • Page d'erreur 404 : La page où atterrissent les utilisateurs qui se perdent.
    • Page des articles : L'URL où s'afficheront vos blogs.
  3. Sauvegardez ! C'est gagné. Allez admirer votre site en direct ! 🚀
